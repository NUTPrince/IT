# 低压
- 50 ma -> 生命危险
- 一个心跳周期  -> 心室颤动
- 摆脱电流：交流 -> 10 ma ，直流 -> 25 ma；人能自行摆脱带电体的 -> 最大电流；男性 -> 9 ma，女性 ->6 ma
- 感知电流：男性 -> 1.1 ma，女性 -> 0.7 ma ｜ 随交流频率增加，感知电流也增加
- 事故多发 -> 用电高峰：6-9月 
- 1 min 内抢救 -> 80%可活
- 高处触电 -> 采取预防措施 ｜ 禁止用手触摸（未脱离电源）
- 断落高压电 -> 8-10 m外（防止跨步电压）
- 触电者脱电后：神智清醒 -> 禁止站立或走动 ｜神智不清 -> 10 s 内判断呼吸心跳情况；若心跳停止  -> 胸外按压，若心跳呼吸都停止 -> 心肺复苏 -> 仰卧，吹气 2 s；按压频率 -> 100次/min；按压与呼吸比例 -> 15 : 1（30 : 2）
- 电杆作业 -> 禁止摇晃；脚扣 -> 短时间使用；安全用具使用完毕  -> 擦拭干净
- 绝缘手套/鞋 -> 半年 ｜ 高压验电器 -> 六个月
- 安全腰带静压力 -> 2206 N
- 标示牌 -> 绝缘材料
- 检修线路 -> 悬挂：禁止合闸，线路有人工作｜变压器梯子 -> 禁止攀爬，高压危险
- 安全标识设计牌是由 -> 安全色、文字、图形和符号所构成；分为 -> 提示、允许、警告、禁止四类
- 电击（内伤） > 电伤（外伤）；直接电击 > 间接
- 电烙印是指 -> 电流化学效应和机械效应 而产生的电伤
- 心室颤动的 -> 最小电流值 作为致命电流界限
- 根据 I=U/R通过人体电流取决于 -> 外加电压与人体电阻（1000-2000 Ω「220 v时」），不高于36V的电压才是安全的
- 为解救触电者而断开电源，事后必须 -> 立即报告上级
- 不属于高压基本绝缘安全用具 -> 绝缘手套、绝缘靴、绝缘垫（属于辅助安全用具）
	- ---> 高压设备的基本绝缘安全用具有：绝缘棒、绝缘夹钳和高压试电
- 绝缘夹钳是在带电的情况下用来安装或拆卸 -> 高压熔断器（35 kv以下）
- 绝缘靴  -> 跨步电压｜绝缘手套 -> 10 cm， -> 相对湿度50～80%储存，6个月进行预防实验
- 绝缘棒验电  -> 35 kv（30kv以上）
- 高压验电器的发光电压不应高于额定电压的  -> 25%
- 10 kv  -> 0.7m以上
- 低压测电笔  -> 交流 60-500
- 停电，邻近高压设备产生  -> 感应电压
- 踏板登杆离地面  -> 15~20cm
- 工作票签发  -> 生产领导人或电气技术负责人
- 操作票  -> 操作人填写；保存  -> 三个月
- 检查现场安全措施 / 工作监护人  -> 工作负责人
- 接地线  -> 摆动，采取相应措施
- 便携接地线  -> 25mm2｜  -> 短路电流
- 现场设备和实际运行方式  -> 相符 的系统模拟图 ｜无法  -> 观察，间接方式确认到位
- 高压室的钥匙有  -> 三把，1.紧急时使用，2. 值班人员使用，3. 巡视和工作负责人
- 重复接地三处不超过  -> 30 Ω
- 中性线对地电压不超过  -> 50 v
- 剩余电流动作保护装置  -> 1000V以下
